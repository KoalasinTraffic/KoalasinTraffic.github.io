# Bandit Enemy

The bandit has a speed aura that can be temporarily removed with the slow tower. If not slowed, all enemies in the area will be given a temporary speed boost.

Speed boost: 40

Speed frequency: 2 seconds

Health: 80%

Movement Speed: 111%
